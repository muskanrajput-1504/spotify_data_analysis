# My First Python Data Project: Spotify Top Artists 🎵📊

Hi! I'm a 4th-semester B.Tech student, and I'm on a mission to become a Data Analyst. 

I recently realized that if I want to land a solid internship in my 3rd year, I need to stop just watching coding tutorials and start actually building things. So, I committed to a daily study routine to learn Python from scratch, and this is my very first hands-on project!

## What is this?
Instead of doing a generic practice exercise, I wanted to work with real data. I downloaded a massive spreadsheet of Spotify songs and used Python to answer one simple question: 

**"Which 10 artists have the most hit songs in this dataset?"**

## How I built it
I kept my setup simple and focused on the core tools of data analysis:
* **Python** (The coding language)
* **Pandas** (A library to open, read, and filter the spreadsheet)
* **Matplotlib** (A library to draw the bar chart)
* **Google Colab** (The browser-based environment where I wrote the code)

## 🚀 What I actually learned doing this
Going from a blank screen to a finished graph taught me a lot. I successfully learned how to:
* Make Python read a raw `.csv` file.
* Count up the data and extract just the top 10 results.
* Format a graph so it actually looks clean (like rotating the artist names so they don't overlap!).
